# Screenshots Directory

This directory contains feature screenshots for the landing page carousel.

## Required Screenshots

Add the following screenshot images to this directory:

1. **dashboard.png** - Screenshot of the main dashboard showing key metrics and overview
2. **properties.png** - Screenshot of the properties management page
3. **tenants.png** - Screenshot of the tenant management interface
4. **invoicing.png** - Screenshot of the invoicing system
5. **maintenance.png** - Screenshot of the maintenance request tracking
6. **leases.png** - Screenshot of the lease management page
7. **leads.png** - Screenshot of the leads pipeline
8. **financials.png** - Screenshot of the financial dashboard

## Image Requirements

- **Format**: PNG or JPG
- **Recommended Size**: 1920x1080 or similar aspect ratio
- **File Size**: Optimize images to keep file sizes reasonable (< 500KB per image recommended)
- **Naming**: Use exact filenames as listed above (case-sensitive)

## How It Works

When a user clicks on a feature card in the carousel, a modal popup will display:
- The feature screenshot (full size)
- A detailed description of the feature
- The feature icon and title

If a screenshot is missing, a placeholder will be shown with instructions.
